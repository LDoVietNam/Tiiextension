# GPT Custom Action

Use `chatgpt-action.yaml` as the Custom Action schema for GPTs.

1. Start Tiiextension local API on `127.0.0.1:18401`.
2. Start Cloudflare Tunnel.
3. Replace `https://YOUR-CLOUDFLARE-TUNNEL.trycloudflare.com` with your tunnel URL.
4. Configure bearer auth with the local API token generated by the installer.

The schema exposes health, tools, workspaces, direct tool calls, agent goals, and event streaming. It does not expose or proxy ChatGPT cookies/session tokens.
